# 300 Calorie Greek Yogurt Berry Chia Bowl

A sweet, high-protein snack using Greek yogurt instead of cottage cheese.

## Ingredients

- 250 g plain 2% Greek yogurt
- 150 g frozen berries
- 10 g chia seeds

## Instructions

1. Add the Greek yogurt to a bowl.
2. Add the frozen berries.
3. Sprinkle the chia seeds on top.
4. Let sit for a few minutes if the berries need to soften, then eat.

## Notes

- This is close to the cottage cheese berry chia bowl, but the texture and flavor are different enough to be useful for variety.
- Do not add honey by default; the berries should be enough sweetness for the standard version.
- If this feels too tart, let the berries thaw longer so their juice mixes through the yogurt.
