# 300 Calorie Cottage Cheese Berry Chia Bowl

A sweet, high-protein, high-fiber snack that works well as an evening snack or a lower-effort afternoon snack.

## Ingredients

- 200 g 2% cottage cheese
- 150 g frozen berries
- 8 g chia seeds

## Instructions

1. Add the cottage cheese to a bowl.
2. Add the frozen berries.
3. Sprinkle the chia seeds on top.
4. Let sit for a few minutes if the berries need to soften, then eat.

## Notes

- This is intentionally simple and does not need honey or other sweetener.
- If using a very tart berry mix, let the berries thaw longer so their juice mixes into the cottage cheese.
- This is a good evening option because it is high-protein without being very heavy.
